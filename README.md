# Task Management System

A full-stack Task Management application built with Django for backend and HTML/CSS/Tailwind for frontend. Users can create, assign, update, and manage tasks with role-based access.

## Tech Stack
- Backend: Django
- Frontend: HTML, CSS, Tailwind CSS
- Database: SQLite / PostgreSQL
- Tools: Git, GitHub

## Features
- User authentication and role-based permissions (Admin, Manager, User)
- CRUD operations for tasks
- Task assignment and status tracking
- Fully responsive frontend using Tailwind CSS
- Fixtures for demo data
